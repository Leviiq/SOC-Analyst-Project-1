# SOC Analytic

## Overview
This repository demonstrates hands-on cybersecurity incident response and SOC analysis through real-world style investigations.

It includes structured case studies covering phishing attacks and malware analysis scenarios.

---

## Case Studies

### 1. Phishing Email Investigation
- Email analysis and header inspection
- Suspicious URL identification
- Phishing indicators detection
- MITRE ATT&CK mapping

### 2. DanaBot Malware Investigation
- Process tree analysis
- Network traffic inspection
- Command and Control (C2) detection
- IOC extraction and correlation

### 3. Red Stealer Malware Investigation
- Malicious process behavior analysis
- Data exfiltration indicators
- Network communication analysis
- Threat intelligence mapping

## Skills Developed 

### 1. Technical Skills:

- Alert triage and classification

- SIEM navigation and log correlation
- IOC identification and validation
- Threat intelligence platform usage

### 2. Frameworks & Methodologies:

- Cyber Kill Chain
- MITRE ATT&CK Framework (basic tactics/techniques)
- Incident response phases (detection, analysis, containment)
 
### 3. Soft Skills:

- Time management under alert load
- Decision-making under uncertainty
- Technical writing (reports, escalations)
- Pattern recognition and critical thinking
